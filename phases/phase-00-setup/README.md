# Phase 00: Setup and Mindset

## Goals
- Install and verify Go toolchain
- Initialize a module
- Run and format a simple Go program

## Theory: Documentation Chapters
- Go installation and setup: https://go.dev/doc/install
- Tutorial chapter "Get started": https://go.dev/doc/tutorial/getting-started
- Go Tour chapters "Welcome", "Basics", "Packages": https://go.dev/tour/
- Modules reference chapter: https://go.dev/ref/mod
- `go` command docs: https://pkg.go.dev/cmd/go

## Checklist
- Run `go version`
- Run `go mod init go-zero-to-hero`
- Create `hello.go` and run `go run hello.go`
- Run `go fmt ./...`

## Extra Practice Projects
- "Go Environment Checker" CLI that prints OS, Go version, GOPATH, and module mode
- "Command Cheatsheet" CLI that prints common Go commands from a local text file
- "Project Bootstrapper" script that creates a starter Go module with folders and README

## Done When
- You can create a new Go file and run it successfully
